liquidGetSloInfo: Command Line Reference

liquidGetSloInfo is a mel command implemented by liquid to allows access to compiled shader information.  It can be used to query shader names, parameters names and types and their defaults.  Fluid uses this command for loading and editing shader parameters and it's easily scriptable.

-setShader [ shaderName ] sets the name of the shader to use, use full path name.
-type returns the type of shader, one of surface, light, volume, displacement or unknown.
-name returns the correct shader name.
-numParam returns the number of parameters in the file ( input and output )
-argName [ arg number ] returns the name of that parameter.
-argType [ arg number ] returns the parameters type.
-argDetail [ arg number ] returns the detail type of the parameters, ie. uniform, varying etc.
-argDefault [ arg number ] returns the default value(s) for the parameter as a string array.
-argExists [ arg name ] returns ture or false whether or not that parameter is availalbe in the specified shader.
-debug this must be put before any of the other options, if used it will print error information to the console.
