Liquid: Feature List

Geometry Output
    -Nurbs Surfaces
    -Nurbs Curves ( as RiCurve and RiNuCurve in RenderDotC )
    -Polygonal Mesh
    -Subdivision Surfaces ( still in development )
    -Particles ( using RiPoints for point particles, RiCurves for lines and RiImplicit for blobbies )
    -Full Trimmed Surfaces

Light Output
    -All Maya lights supported
    -If no shader is attached to the lights than default renderman shaders are used ( pointlight, spotlight, distantlight and ambientlight )
    -Automatic shadow generation for spotlights, directional lights and point lights
    -Shadow Camera controllers can be used to interactive place shadows
    -Initial support for Area Lights.

Texture Output
    -Full U/V Support for Nurbs Surfaces
    -Full U/V Support for Nurbs Curves
    -Full U/V Support for Polygonal Meshes
    -Full U/V Support for Subdivision Surfaces
    -Particles can have U/V set with Maya particle expressions
    -Alternative output for Polygonal Meshes to support multiple U/V sets
    -Normalized U/V output for Nurbs Surfaces

Custom Attributes
    -Nurbs, Polygons, Subdivision Surfaces and Lights can all have custom attributes attached that get passed into renderman on the parameter list
    -Support for Float, Float Array, String, Point, Point Array, Color, Color Array, Normal, Normal Array, Vector, Vector Array

Shader Interface
    - Fluid, a full featured shader interface
    - Tools include load, reload, remove, duplicate, edit parameters, assign, pick assigned objects, pick assigned shader and preview
    - Support for Liquid Interface Files ( .lif ) to build custom dialogs for all shader types to include help, drop down menus and set default parameters
    - Multiple selection, assigning and loading
    - Modal interface
    - Shader preview window with auto-update option when shader parameters change

Scripting, Expressions and Devkit
    - Fully integrated into Maya to use all of Maya's expression tools
    - Long list of commands for easy mel scripting
    - Command line version for distributed rib gen and unix scripted rib generation
    - Support for procedural Rib Genetator plugins with full to the Maya scene and Renderman bindings in the Developers Kit
    - Ships with Developers Kit.
