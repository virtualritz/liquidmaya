Liquid: Installation

Liquid can be installed just like any other Maya plug-in. Once you have untarred/unzipped ( depending on which platform you are using ) Liquid into the directory where you want it you have to set a few environment variables that Maya will use to find it.

The first one will point to the location of Liquids mel directory, add

IntstallationDirectory\mel to the MAYA_SCRIPT_PATH variable.

then add

InstallationDirectory\bin to the MAYA_PLUG_IN_PATH variable, this will ensure maya will be able to find the plugin when loaded. Lastly add

InstallationDirectory\bin to your path, this will ensure the deferred rib generator renders will be able to find liquid.
