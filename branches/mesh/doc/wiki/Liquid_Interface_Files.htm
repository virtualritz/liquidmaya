Liquid Interface Files

From LiquidWiki

Liquid Interface Files (LIF) describe a shader's parameters to make them more usable in the Fluid editing window.

Shader parameters can be:

  • Labeled with a human-readable name.
  • Limited in range.
  • Re-ordered in the editing window.
  • Organized in groups.
  • Set to use a more appropriate widget than the default one.
  • Described to make TDs happy.

[edit]

Keywords

surface "shadername" 
This is a surface shader
light "shadername" 
This is a light shader
displacement "shadername" 
This is a displacement shader
volume "shadername" 
This is a volume shader
param "paramname" 
Name a of a shader parameter
name "label" 
The GUI label for the shader parameter ( optional )
desc "descriptions" 
A Description of the shader parameter ( optional )
type "widget" 
The type of widget you want for this parameter ( optional )
range [values...] 
The recommended range of this parameter ( optional ). A default value may be specified for the range.
channelbox "on|off" 
make the param appear in the channel box or not ( optional )
default : on
group "name"/"end" 
Starts/Ends a parameter group.
In its opening form, group accepts an optional "open" or "close" attribute to specify its default state.
[edit]

Widget Types

The default widget type is automatically selected based on the parameter's type, but you can override it in a lif file.

The type keyword supports a number of keywords :

[edit]

slider

a simple slider with an optional [min max] range. 

param "filterwidth";
name  "Filter Width";
type  "slider";
range [0 1]
[edit]

bool

a simple check box. 

param "frontFacesOnly";
type  "bool";
[edit]

selector

an option menu. values must be specified with range. 

param "mode1";
name  "Mode 1";
type  "selector";
range [Normal 0 Special 1 "Top Secret" 2]
default "1";
[edit]

texture

a text field with a button to open a texture browser. 

param "mapName1";
name  "Map Name 1";
type  "texture";
[edit]

shadow

a text field with a button to open a shadow texture browser. 

param "shadowfile";
type  "shadow";
[edit]

coordsys

a text field with a button to open a coordinate systems browser. 

param "envcoordsys";
type  "coordsys";
[edit]

colorspline

A gradient control to hook up with your color array. 

# simple gradient, fixed position, fixed interpolation
 
paramname "myColorSpline";
type "colorspline myColorSpline"

# gradient with position control, fixed interpolation

paramname "myColorSpline";
type "colorspline myColorSpline myColorPositions"

# gradient with position and interpolation control
# the interp variable can be an array or not.

paramname "myColorSpline";
type "colorspline myColorSpline myColorPositions myColorInterpolation"
[edit]

floatspline

A gradient control to hook up with your color array. 

# simple gradient, fixed position, fixed interpolation
 
paramname "myFloatSpline";
type "floatspline myFloatSpline"

# gradient with position control, fixed interpolation

paramname "myColorSpline";
type "colorspline myFloatSpline myFloatPositions"

# gradient with position and interpolation control
# the interp variable can be an array or not.

paramname "myColorSpline";
type "colorspline myFloatSpline myFloatPositions myFloatInterpolation"

A typical Lif file

light "liquidspot";
desc "Standard Maya-Like Spot Light\nPart of Liquid.";

param "intensity";
name "intensity";
desc "";
type "slider";
range [ 0 1 ];

param "lightcolor";
name "lightcolor";
desc "";

param "__nondiffuse";
name "Non Diffuse";
desc "";
type "bool";

param "__nonspecular";
name "Non Specular";
desc "";
type "bool";

param "coneangle";
name "coneangle";
desc "";
type "slider";
range [ 0 1 ];

param "penumbraangle";
name "penumbraangle";
desc "";
type "slider";
range [ 0 1 ];

param "dropoff";
name "dropoff";
desc "";
type "slider";

param "decay";
name "decay";
desc "";
type "selector";
range [ None 0 Linear 1 Quadratic 2 Cubic 3 ];

group "Barn Doors";
	param "barndoors";
	name "barndoors";
	desc "";
	type "bool";

	param "leftbarndoor";
	name "leftbarndoor";
	desc "";
	type "slider";
	range [ 0 1 ];

	param "rightbarndoor";
	name "rightbarndoor";
	desc "";
	type "slider";
	range [ 0 1 ];

	param "topbarndoor";
	name "topbarndoor";
	desc "";
	type "slider";
	range [ 0 1 ];

	param "bottombarndoor";
	name "bottombarndoor";
	desc "";
	type "slider";
	range [ 0 1 ];
group "end";

group "Shadow";
	param "shadowname";
	name "Shadow Name";
	type "shadow";

	param "shadowbias";
	name "Shadow Bias";
	desc "";
	type "slider";
	range [ 0 1 ];

	param "shadowblur";
	name "Shadow Blur";
	desc "";
	type "slider";
	range [ 0 1 ];

	param "shadowsamples";
	name "Shadow Samples";
	desc "";
	type "selector";
	range [ 8 8 16 16 32 32 64 64 ];

	param "shadowfiltersize";
	name "Shadow Filter Size";
	desc "";
	type "slider";
	range [ 0 1 ];

	param "shadowcolor";
	name "Shadow Color";
	desc "";

	group "AOVs" "close";
		param "__shadow";
		name "__shadow";
		desc "";

		param "__unshadowed_Cl";
		name "__unshadowed_Cl";
		desc "";
	group "end";
group "end";
